# WezMyVan
Never miss the ice cream van again — live map, route schedules &amp; pre-orders for Icecream 🍦
Wez Me Van!! is a React Native app for Lancashire that lets customers track local ice cream vans in real time, browse their menus, and pre-order before the van reaches their street. Van operators get a live dashboard to broadcast their GPS, plan daily routes, manage their menu, and receive orders — all from their phone.
Built with Expo, Supabase, Stripe Connect, and RevenueCat. Physical orders are processed through Stripe; the Van Pro subscription tier uses Apple IAP / Google Play Billing via RevenueCat.
